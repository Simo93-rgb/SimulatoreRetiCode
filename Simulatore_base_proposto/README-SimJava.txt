Il programma Java in questa cartella è un esempio di simulatore ad eventi discreti di coda singola. E' tratto dal libro di testo. 
Richiede un argomento, da passare su linea di comando quando lo si lancia: è il seme del generatore di numeri casuali.

Non è un simulatore completo: esegue un solo run (mentre uno studio di simulazione completo deve eseguire più run e applicare metodi statistici ai campioni raccolti da ciascun run), inoltre usa la libreria Random di Java per la generazione dei numeri casuali. Vedremo altre librerie utilizzabili allo scopo.
